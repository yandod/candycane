Technical Info & Requirements

CandyCane is powered by CakePHP 2.1. To run, you need a HTTP server (e.g. Apache) and PHP 5.2.8 or greater. We currently supports MySQL 4 or PostgreSQL X.
For detailed information, please check CakePHP requirements documetation.
